DataMind AI is an AI-powered data analytics web app I built to solve a common problem: non-technical users struggling to extract insights from raw CSV/Excel data without knowing SQL or coding. I developed it using Streamlit and Google Gemini API, allowing users to simply ask questions in plain English and receive structured tables and statistical summaries in return. I deployed it on AWS EC2, handling Linux server setup, security groups, and port configuration end-to-end. The impact was clear — it removed the technical barrier between users and their data, enabling anyone to analyze datasets and download results without writing a single line of code.
